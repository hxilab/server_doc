# GPU利用状況を確認する

学習や推論を実行する際にリアルタイムでGPU利用状況を確認できる

以下のコマンドをターミナルで実行
```shell
$ nvidia-smi
```

コマンド実行時のGPU使用率やメモリ使用量，消費電力，温度などが表示される．

また，以下のオプションで```nvidia-smi```コマンドを定期実行でき，利用状況をリアルタイムに把握できる

```shell
$ nvidia-smi -l        # デフォルトは5秒間隔
$ nvidia-smi -l 1      # 1秒間隔
$ nvidia-smi -lms 500  # 0.5秒間隔
```
